How it works

Three steps, that's it

No accounts, no pairing codes, no installs on the receiving end.

1

Run BoltDrop

Launch the app on your Mac, Windows, or Linux machine. It starts a local server instantly.

2

Open the URL

Any device on the same WiFi visits the displayed address in a browser — phone, tablet, laptop.

3

Drop & transfer

Drag files in, hit send. Files transfer at full local network speed with zero internet hops.

Features

Built for speed and privacy

Everything you need for hassle-free local transfers, nothing you don't.

No receiver install

Receiving devices use a plain browser. Nothing to install, no app store, no setup.

100% private

Files never touch the internet. All traffic stays on your local network.

Full network speed

No cloud bottlenecks. Transfers hit your router's ceiling — often 400+ MB/s on WiFi 6.

Cross-platform

Mac, Windows, Linux. Send from any OS, receive on any device with a browser.

No account required

Open source and free. No sign-up, no telemetry, no subscriptions.

Real-time progress

Live transfer progress bar, speed readout, and ETA — visible on both ends.
