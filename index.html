How it works

Three steps, that's it

No accounts, no pairing codes, no installs on the receiving end.

1

Run BoltDrop

Launch the app on your Mac, Windows, or Linux machine. It starts a local server instantly.

2

Open the URL

Any device on the same WiFi visits the displayed address in a browser — phone, tablet, laptop.

3

Drop & transfer

Drag files in, hit send. Files transfer at full local network speed with zero internet hops.

Features

Built for speed and privacy

Everything you need for hassle-free local transfers, nothing you don't.

No receiver install

Receiving devices use a plain browser. Nothing to install, no app store, no setup.

100% private

Files never touch the internet. All traffic stays on your local network.

Full network speed

No cloud bottlenecks. Transfers hit your router's ceiling — often 400+ MB/s on WiFi 6.

Cross-platform

Mac, Windows, Linux. Send from any OS, receive on any device with a browser.

No account required

Open source and free. No sign-up, no telemetry, no subscriptions.

Real-time progress

Live transfer progress bar, speed readout, and ETA — visible on both ends.

Get started

Install & run in seconds

One command installs BoltDrop. Works on Mac, Linux, and Windows.

Step 1 — Install
🍎 macOS
🐧 Linux
# Paste this in Terminal — auto-detects Apple Silicon or Intel curl -fsSL https://raw.githubusercontent.com/boltdrop/boltdrop.github.io/refs/heads/main/install.sh | sh
Installs to /usr/local/bin/boltdrop and removes the macOS quarantine flag automatically.
# Paste this in Terminal — auto-detects x86_64 / arm64 / armv7 curl -fsSL https://raw.githubusercontent.com/boltdrop/boltdrop.github.io/refs/heads/main/install.sh | sh
Installs to /usr/local/bin/boltdrop. Requires sudo for the move.
Step 2 — Start BoltDrop
# Start on default port 3000 boltdrop # Or specify a custom port boltdrop --port 8080 # See all options boltdrop --help
Step 3 — Transfer files
💻
On your computer
Run boltdrop — it prints a local URL like http://192.168.1.12:3000
📱
On your phone / tablet / other device
Open the URL in any browser — no app needed. Both devices must be on the same WiFi.
⚡
Drop & send
Drag files into the browser window and hit Send. Files transfer at full local network speed.
